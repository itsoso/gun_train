# 智能枪械训练系统 - 使用手册

## 一、系统概述

### 1.1 系统简介

智能枪械训练系统是一个基于AI视觉识别技术的枪械训练辅助平台，通过实时监控和分析学员的训练动作，提供即时反馈和规范化指导，帮助学员形成正确的肌肉记忆。

### 1.2 核心功能

1. **空枪训练**：无弹药成本的大量练习
2. **动作规范检测**：AI实时分析动作规范性
3. **即时反馈**：屏幕显示和语音提示
4. **安全预警**：危险动作自动报警
5. **训练数据统计**：个人档案和进度跟踪
6. **教官辅助**：减轻教官压力，提高效率

### 1.3 训练流程

```
第一阶段：空枪规范化训练
├── 基础持枪姿势练习
├── 瞄准动作练习
├── 扳机控制练习
└── 连贯动作练习（需连续5次合格）

第二阶段：空枪训练考核
├── 完成连续5次标准动作
├── 平均分≥80分
└── 通过考核

第三阶段：实弹射击训练
├── 领取弹药（需教官批准）
├── 实弹射击（AI继续监控）
└── 成绩评估和总结
```

## 二、学员端操作指南

### 2.1 登录系统

1. 在学员工位终端上打开浏览器
2. 访问系统地址（如：`http://train.local`）
3. 输入警号和密码登录
4. 系统自动分配工位

### 2.2 开始训练

#### 步骤1：领取空枪

1. 向教官申请领取空枪
2. 教官在系统中分配枪支和工位
3. 确认枪支为空枪（无弹药）

#### 步骤2：调整工位

1. 站在指定工位标记位置
2. 调整显示屏到合适角度
3. 确认摄像头可以完整拍摄到你的动作

#### 步骤3：开始练习

1. 屏幕显示"准备开始"
2. 按照标准动作持枪
3. 系统实时显示你的动作评分和错误提示

### 2.3 理解屏幕显示

#### 评分说明

- **总体得分**：综合评价（0-100分）
  - 90-100分：优秀
  - 80-89分：良好（合格）
  - 70-79分：及格
  - 60-69分：需改进
  - <60分：不合格

- **分项得分**：
  - **持枪姿势**（30%权重）：手臂角度、握持方式、身体姿态
  - **扳机纪律**（30%权重）：手指位置、扳机控制
  - **瞄准线**（20%权重）：眼睛-准星-目标三点一线
  - **稳定性**（20%权重）：手部抖动、重心稳定

#### 错误提示

屏幕会实时显示需要改进的地方：

- 🔴 **红色警告**：严重错误，立即纠正
- 🟡 **黄色提示**：需要注意的问题

常见错误示例：
- "右肘角度过大(125°)，应保持在90-120°"
- "手指未离开扳机，违反扳机纪律"
- "头部位置过低，无法形成正确瞄准线"
- "手部抖动明显，注意稳定性"

### 2.4 训练技巧

#### 持枪姿势要点

1. **握枪**：
   - 右手虎口紧握枪柄
   - 左手从下方托住枪身
   - 双手用力均匀

2. **手臂**：
   - 右臂肘部弯曲90-120度
   - 左臂自然支撑，不要过直
   - 肩部放松，不要耸肩

3. **身体**：
   - 双脚与肩同宽
   - 身体微前倾
   - 重心在前脚掌

#### 扳机纪律

**核心原则**：非射击时手指绝对不能接触扳机！

- ✅ **正确**：食指伸直，沿枪身放置
- ❌ **错误**：食指放在扳机上

#### 瞄准要领

1. 双眼看向目标
2. 枪口准星对准目标
3. 保持眼睛-准星-目标三点一线
4. 头部保持稳定，不要歪头

#### 稳定性训练

1. 深呼吸，保持放松
2. 扣压扳机时要平稳
3. 避免突然用力
4. 多练习，形成肌肉记忆

### 2.5 进度跟踪

屏幕下方显示你的训练进度：

```
训练进度 - 空枪训练阶段
■■■□□ 3/5 次合格
继续保持！连续5次达标后即可进入实弹训练
```

- 绿色进度条：表示合格次数
- 需要连续5次≥80分才能通过空枪训练

### 2.6 注意事项

⚠️ **安全第一**

1. **枪口方向**：只能指向靶区或地面，绝不指向人
2. **扳机纪律**：非射击时手指必须离开扳机
3. **听从指挥**：遵守教官指令和安全规范
4. **及时报告**：发现异常立即报告教官

⚠️ **系统提示**

- 如遇到严重警告（红色闪烁），立即停止动作
- 系统会自动记录所有违规行为
- 多次严重违规可能被取消训练资格

## 三、教官端操作指南

### 3.1 登录和监控

1. 使用教官账号登录系统
2. 进入"监控中心"页面
3. 可同时查看所有工位的实时画面

### 3.2 创建训练场次

#### 步骤：

1. 点击"新建场次"
2. 填写信息：
   - 训练日期：选择日期
   - 训练类型：空枪训练/实弹训练
   - 参训人员：选择学员
3. 点击"开始训练"

### 3.3 分配工位

1. 在学员列表中选择学员
2. 点击"分配工位"
3. 选择工位号（1-50）
4. 选择枪支型号
5. 确认分配

### 3.4 实时监控

#### 多工位视图

- 屏幕同时显示所有活跃工位
- 绿色边框：正常训练
- 黄色边框：有需要注意的问题
- 红色边框：严重警告，需立即处理

#### 单工位详情

点击任意工位，查看详细信息：
- 学员姓名、警号
- 实时视频（3个角度）
- 当前得分和错误列表
- 训练进度
- 历史记录

### 3.5 处理预警

系统会自动标注问题学员：

1. 点击红色预警标识
2. 查看具体问题
3. 决定处理方式：
   - 远程语音指导
   - 现场指导
   - 暂停训练
4. 点击"已处理"记录

### 3.6 批准实弹训练

当学员完成空枪训练考核后：

1. 系统自动通知"XX学员已通过考核"
2. 点击查看学员详情
3. 确认：
   - 连续5次合格
   - 平均分≥80
   - 无严重违规记录
4. 点击"批准实弹训练"
5. 发放弹药并登记

### 3.7 弹药管理

#### 发放弹药

1. 进入"弹药管理"页面
2. 选择学员
3. 选择弹药类型和数量
4. 确认发放
5. 学员签字确认

#### 回收弹药

1. 训练结束后清点弹药
2. 录入：
   - 使用数量
   - 归还数量
3. 核对无误后确认

### 3.8 生成报告

训练结束后：

1. 点击"结束训练"
2. 系统自动生成报告：
   - 参训人员统计
   - 成绩分布
   - 预警情况
   - 弹药使用情况
3. 导出PDF或Excel

### 3.9 教官工作技巧

#### 高效监控

- 重点关注红色和黄色标记的工位
- 利用系统自动筛选功能
- 定期查看整体统计数据

#### 个性化指导

- 查看学员历史数据
- 针对薄弱环节重点指导
- 鼓励进步明显的学员

#### 安全管理

- 定期巡视现场
- 不完全依赖系统，保持警觉
- 发现危险行为立即制止

## 五、常见问题解答

### Q1: 系统提示"无法检测到姿态"怎么办？

**A**: 
1. 检查摄像头是否能完整拍摄到你
2. 确保光线充足
3. 不要穿着反光材质的衣物
4. 联系技术人员检查摄像头

### Q2: 为什么我的分数总是不够80分？

**A**: 
1. 仔细查看错误提示，针对性改进
2. 可以点击"查看标准示范"
3. 请教官现场指导
4. 多练习，慢慢形成正确动作

### Q3: 实弹训练需要多久才能参加？

**A**: 
- 需要完成空枪训练并通过考核
- 要求：连续5次动作≥80分
- 平均分≥80分
- 无严重违规记录
- 通常需要练习50-100次

### Q4: 系统突然提示"危险警告"是什么意思？

**A**: 
- 说明你的动作存在严重安全隐患
- 最常见：枪口指向了非安全区域
- 立即停止动作，检查枪口方向
- 确保枪口只指向靶区或地面

### Q5: 训练数据保留多久？

**A**: 
- 训练记录永久保存
- 视频录像保留30天
- 重要视频（如优秀示范、违规行为）长期保留
